# Tetris CLI

Классический Тетрис прямо в вашем терминале. Никаких зависимостей,

только Python 3 и несколько нажатий клавиш. Работает на macOS, Linux и Windows.

Написано на Python 3 · Использует curses · Открытый исходный код

macOS
Python 3 уже установлен на macOS. Просто откройте Терминал.
В команде cd укажите путь к папке, в которую вы сохранили tetris.py.
Например, если скачали в Загрузки — это ~/Downloads.
Если сохранили на Рабочий стол — используйте ~/Desktop.
% cd /папка/где/лежит/tetris.py
% python3 tetris.py
Чтобы открыть Терминал: Spotlight (⌘ Space) → напишите «Terminal» → Enter
🐧 Linux
Если Python 3 не установлен — поставьте его через пакетный менеджер.
Замените /папка/где/лежит/tetris.py на реальный путь к файлу.
Обычно это ~/Downloads (Загрузки) или ~/Desktop (Рабочий стол).
$ sudo apt install python3 # если ещё нет
$ cd /папка/где/лежит/tetris.py
$ python3 tetris.py
$ sudo pacman -S python
$ cd /папка/где/лежит/tetris.py
$ python3 tetris.py
🪟 Windows
На Windows нужно установить Python и дополнительный пакет windows-curses.
Замените C:\путь\к\папке на папку, в которую сохранили файл.
Обычно это $HOME\Downloads (Загрузки) или $HOME\Desktop (Рабочий стол).
1. Скачайте Python 3 с python.org/downloads
2. ☑ Поставьте галку «Add Python to PATH»
PS> pip install windows-curses
PS> cd C:\путь\к\папке\с\tetris.py
PS> python tetris.py
Чтобы открыть PowerShell: Win + R → напишите «powershell» → Enter
Или нажмите Win + X → «Windows PowerShell»
